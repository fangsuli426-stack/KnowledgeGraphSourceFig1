项目简介：
本研究提出了一种基于改进YOLOv5的番茄检测与分类模型——CAM-YOLO，通过引入组合注意力机制（Combining Attention Mechanism）来提升番茄目标的检测精度和分类性能。该模型在复杂农业环境下表现出优异的识别能力，为精准农业和自动化采收提供了有效的技术解决方案。
